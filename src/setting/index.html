Status Number (Status Badge)

On the top-right of the extension icon is the status number, which is used to display what has been matched in the current tab (the tab ID, the tab URL, etc.) to stop the URL that matched from getting saved.

The position of the digit showed what the extension matches in the current URL/tab, while the actual digit showed if the condition was met (Number 0 means NOT MATCHED, while number 1 means MATCHED)

Here's a reference table to decode what does the number means:

Digit positioned # in the status number What it mached
#1 The URL in the current tab.
#2 The current tab ID.
#3 The current page's title.
